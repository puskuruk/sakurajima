# Format Mac (Clean Slate)

## Before Erasing

1. Verify Time Machine backup is complete
2. Sign out: System Settings → Apple ID → Sign Out
3. Disable Find My Mac

## Erase (Apple Silicon / T2)

System Settings → General → Transfer or Reset → **Erase All Content and Settings**

## Erase (Older Intel)

1. Restart holding `⌘R`
2. Disk Utility → Erase Macintosh HD (APFS, GUID)
3. Reinstall macOS

## After Format

1. Create admin user (don't restore from backup)
2. Install all macOS updates
3. Continue to [INSTALL.md](INSTALL.md)
