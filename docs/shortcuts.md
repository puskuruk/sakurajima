# Shortcuts

## AeroSpace (Window Manager)

### Focus/Move/Swap
| Action | Focus | Move | Swap |
|--------|-------|------|------|
| Left | `⌥H` | `⌥⇧H` | `⌃⌥H` |
| Down | `⌥J` | `⌥⇧J` | `⌃⌥J` |
| Up | `⌥K` | `⌥⇧K` | `⌃⌥K` |
| Right | `⌥L` | `⌥⇧L` | `⌃⌥L` |

### Layout
| Action | Key |
|--------|-----|
| Toggle orientation | `⌥/` |
| Toggle fullscreen | `⌥F` |
| Balance windows | `⌥⇧=` |
| Resize mode | `⌥R` |

### Workspaces
| Workspace | Switch | Move |
|-----------|--------|------|
| Terminal | `⌥1` | `⌥⇧1` |
| IDE | `⌥2` | `⌥⇧2` |
| Web | `⌥3` | `⌥⇧3` |
| Messages | `⌥4` | `⌥⇧4` |
| Clients 1-9 | `⌃⌥⌘1-9` | `⌃⌥⌘⇧1-9` |

### Navigation
| Action | Key |
|--------|-----|
| Previous workspace | `⌥[` |
| Next workspace | `⌥]` |
| Last workspace | `⌥\`` |

## CLI Commands

| Command | Description |
|---------|-------------|
| `focus <client>` | cd + workspace + time tracking + SSH |
| `unfocus` | Stop time tracking |
| `skr new <client>` | Create client workspace |
| `skr stats` | View time stats |
| `skr verify` | System health check |
| `skr infra up/down` | Docker infrastructure |

## Quick Reference

```
FOCUS      ⌥ + H/J/K/L
MOVE       ⌥⇧ + H/J/K/L
WORKSPACE  ⌥ + 1/2/3/4
CLIENT     ⌃⌥⌘ + 1-9
RESIZE     ⌥R then H/J/K/L
```
