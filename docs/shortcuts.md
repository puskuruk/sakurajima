# Sakurajima Keyboard Shortcuts

A quick reference for window management and launch bindings across AeroSpace, Rectangle, and Raycast.

## AeroSpace (Tiling WM)
Source: `~/setup/configs/aerospace.base.toml`

| Action | Shortcut |
| --- | --- |
| Focus window left / down / up / right | ⌥H / ⌥J / ⌥K / ⌥L |
| Move window left / down / up / right | ⌥⇧H / ⌥⇧J / ⌥⇧K / ⌥⇧L |
| Resize width −50 / +50 | ⌃⌥⌘H / ⌃⌥⌘L |
| Resize height −50 / +50 | ⌃⌥⌘K / ⌃⌥⌘J |
| Toggle horizontal/vertical tile layout | ⌥⏎ |
| Toggle fullscreen | ⌥F |
| Switch to workspace T / I / W / M | ⌥1 / ⌥2 / ⌥3 / ⌥4 |
| Move window to workspace T / I / W / M | ⌥⇧1 / ⌥⇧2 / ⌥⇧3 / ⌥⇧4 |
| Client workspaces 1–9 (generated) | ⌃⌥⌘1…9 |
| Move window to client workspace 1–9 | ⌃⌥⌘⇧1…9 |

## macOS Shortcuts (Native Window Snapping)
Generated by `install-aerospace-native-shortcuts` into:
`~/.local/share/sakurajima/aerospace-actions/`

Assign these in **Shortcuts.app** (Run Shell Script → `/bin/zsh` → call the script path):

| Action | Shortcut | Script |
| --- | --- | --- |
| Left/Right/Top/Bottom half | ⌃⌥← / ⌃⌥→ / ⌃⌥↑ / ⌃⌥↓ | `left-half.sh` etc. |
| Corners | ⌃⌥U / ⌃⌥I / ⌃⌥J / ⌃⌥K | `top-left.sh` etc. |
| Maximize | ⌃⌥⏎ | `maximize.sh` |
| Maximize height | ⌃⌥⇧↑ | `maximize-height.sh` |
| Smaller / Larger | ⌃⌥- / ⌃⌥= | `smaller.sh` / `larger.sh` |
| Center / Restore | ⌃⌥C / ⌃⌥⌫ | `center.sh` / `restore.sh` |
| Open Terminal | ⌃⌥T | `open-terminal.sh` |

## Raycast Commands
Generated via `generate-sakurajima-commands.sh` → `~/raycast/scripts/sakurajima`

| Command | Shortcut | Description |
| --- | --- | --- |
| Sakurajima: Open Terminal | ⌃⌥T | Launches a fresh Terminal window |
| Sakurajima: Update AeroSpace | — | Regenerates & reloads AeroSpace config |
| Sakurajima: Verify System | — | Runs Sakurajima health checks |
| Sakurajima: Infra Up / Down | — | Start/stop local Docker stack |
| Sakurajima: Focus <client> | assign per Raycast | Opens Terminal in client workspace |
| Sakurajima: Focus {client-name} | assign per Raycast | Client-specific focus launchers |

> Tip: Adjust Raycast hotkeys in **Raycast → Settings → Extensions** if you prefer different bindings.
